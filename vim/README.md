# Grimm VIM
My vimrc is fairly commented and hopefully understandable.  There is commented out stuff from old settings that I don't use anymore or that are covered by plugins that I now use.
I've included a `setup_vim_plugins.sh` script to install and/or update the plugins that I use.

## Let's do this
1. Copy the .vimrc and .vim/ to your `$HOME`
1. execute `setup_vim_plugins.sh`
1. Bask in the glory of [VIM](https://www.vim.org)
